# Smart Retail AI System

Smart Retail AI System is a web-based retail management project designed to help retailers manage products, customers, carts, orders, invoices, stock, and admin dashboard insights in a smarter and more organized way.

## Features

### Customer Side
- User authentication
- Product browsing
- Add to cart
- Checkout system
- Order placement
- Invoice generation
- Product recommendations
- Responsive user interface

### Admin Side
- Admin dashboard
- Add, update, and delete products
- View total products
- View total orders
- Track total revenue
- Low stock product alerts
- Best-selling product analysis
- Sales insight summary

### Smart Features
- Basic product recommendation system
- Smart product description generation
- Sales insights
- Low-stock detection

## Tech Stack

- HTML
- CSS
- JavaScript
- Firebase Authentication
- Firebase Firestore
- Firebase Hosting
- jsPDF
- html2canvas

## Project Purpose

The main purpose of this project is to help small retailers manage their business digitally. It reduces manual work and provides useful business insights through a simple admin dashboard.

This project was built as a practical learning project to understand real-world retail workflows, database handling, authentication, order management, invoice generation, and inventory tracking.

## Admin Dashboard Insights

The admin dashboard shows:

- Total products
- Total orders
- Total revenue
- Low stock products
- Most sold product
- Top selling products
- Sales summary

## Invoice System

The system generates invoices for customer orders using jsPDF and html2canvas.

Invoice includes:

- Customer details
- Product details
- Quantity
- Price
- Total amount
- Order date
